# SQX Tool - StrategyQuant X Project Helper

Una herramienta de línea de comandos para automatizar la creación y gestión de proyectos de StrategyQuant X.

## 📥 Descarga

La última versión del SQX Tool está disponible para descargar desde:

**https://github.com/ivanrdgc/sqx-tool/releases**

Descarga el archivo ZIP de la última versión y descomprímelo.

## 🚀 Instalación y Uso

### Paso 1: Descomprimir
Descomprime el archivo ZIP **justo dentro de la carpeta de instalación de StrategyQuant X**.

```
StrategyQuant/
├── sqx-tool/
│   ├── src/
│   ├── Projects/
│   ├── run.bat
│   └── run.sh
└── ... (otros archivos de StrategyQuant)
```

### Paso 2: Ejecutar
Ejecuta el archivo **`run.bat`** (en Windows) o **`run.sh`** (en Linux/Mac) para iniciar la herramienta.

### Paso 3: Cargar Proyecto
1. Abre StrategyQuant X
2. Carga el proyecto personalizado desde la carpeta **`Projects`**
3. Arranca el proyecto en StrategyQuant X

## 📋 Características

- Creación automática de proyectos SQX desde plantillas
- Configuración automática de símbolos, timeframes y direcciones
- Gestión de comisiones y swaps
- Creación de proyectos desde plantilla Ramon Mensual
- Eliminación automática de ExitAfterBars
- Renombrado automático de archivos según convenciones

## 🔧 Requisitos

- StrategyQuant X instalado
- Base de datos de símbolos configurada
- Python 3.x (incluido en la distribución)
